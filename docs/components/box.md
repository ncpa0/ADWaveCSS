# Box

Box is a simple container element. By default comes with a flex display type.

```html
<div class="box">
  <!-- content -->
</div>
```

## Brighter box

Box elements when the `light` class name is assigned to them change their background color to a brighter one.

```html
<div class="box light">
  <!-- content -->
</div>
```
